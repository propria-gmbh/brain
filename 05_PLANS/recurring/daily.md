# Daily Shopify Checklist

- [ ] Разобрать почту

- [ ] Запросить реальные фото для новых товаров
- [ ] Оплатить инвойсы поставщикам
- [ ] Сохранить инвойсы поставщика
- [ ] Проверить тикеты службы поддержки (Gorgias)
- [ ] Проверить GAds
- [ ] PNL — заполнить (еженедельно)
- [ ] Запросить квоты на новые товары у 3PL
- [ ] Ответить AB в Slack

## Контейнеризация дня

- 1ая половина: работа, задачи продвигающие к целям, обучение
- 2ая половина: технические задачи, платежи, задачи по дому
- Перекур / На улице: звонки, короткие письма, прогулка — единый контейнер
- Растяжка: 5 мин в конце каждого часа или 10 мин в конце каждых 2 часов
