# Чеклист верификации существующего домена

Применять для каждого домена магазина перед настройкой GMC.

- [ ] RDAP: дата создания > 1 года, непрерывность (нет gap)
- [ ] Privacy OFF — компания видна в WHOIS
- [ ] HTTPS + SSL (200 ответ)
- [ ] ScamAdviser: score ≥ 75 (если < 75 — проверить Trustpilot)
- [ ] Trustpilot: нет негативных отзывов / жалоб на домен
- [ ] Google Safe Browsing: нет флага
- [ ] site:домен в Google — 0 результатов
- [ ] Domain authentication в Shopify → Settings → Notifications
