# GMC — знания

## Регистрация

- Регистрировать GMC можно на физика — рабочий вариант
- Для Shopify: реальные данные компании
- Для GMC и сайта: произвольный адрес страны рынка
  - Проверка адреса: нет дропшиперов по этому адресу, результатов поиска не более 1 страницы
- Юрлицо в GMC изменить нельзя. Единственный вариант — удалить payment profile (но тогда всё сначала)
- Передача GMC: передаётся gmail на которой создан GMC + GAds, затем трансфер Shopify магазина со сменой юрлица (Максим Лукьянов, 22.05.26)

## Мультифид (Symprosys)

- Сначала добавить политики в GMC
- Потом добавить новый рынок через Symprosys
