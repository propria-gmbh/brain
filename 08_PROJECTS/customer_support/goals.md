# Goals — customer_support

## 1 месяц

- Draft-only работает стабильно, качество ответов устраивает
- KB пополняется через `/extract-knowledge` после resolutions

## 3 месяца

- Готовность к tiered auto-send (Layers 18-20 активированы для низкорисковых кейсов)
- Метрики: % автодрафтов принятых без правок, время ответа, compliance-нарушений = 0

## 12 месяцев

- Auto-send для стандартных случаев по всем магазинам
- Подключение 3-го магазина без увеличения поддержки
- Proactive outreach (не только reactive)

## Критерии успеха

- `qa-reviewer` блокирует < X% драфтов (калиброваться)
- 0 нарушений GMC / UCPD / statutory rights
- Время от сообщения клиента до отправленного ответа ↓ на Y%
