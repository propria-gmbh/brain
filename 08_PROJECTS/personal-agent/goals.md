# Goals — personal-agent

## 1 месяц

- Стабильная работа, без manual-restart инцидентов
- Memory растёт управляемо (policy: что попадает в `memory/`, что нет)

## 3 месяца

- Интеграция с brain/ — возможность читать `~/Projects/brain/` (read-only) или синхронизированная копия memory с brain/
- TTS качество стабильно

## 12 месяцев

- Voice in (speech-to-text) — полный voice loop
- Агент может делать небольшие автоматические задачи (не только отвечать)

## Критерии успеха

- Uptime > 99%
- Memory файлов < N (чтобы не захламлялось)
- 0 leak секретов
