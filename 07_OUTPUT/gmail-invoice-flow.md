# Флоу работы с инвойсами в Gmail

## Когда пришёл инвойс

1. Добавить метку **3. Payments** (очередь на оплату)
2. Если инвойс касается Propria — добавить ещё **Propria GmbH/Rechnungen**

## В платёжный слот (Вт/Пт 15:00)

3. Открыть папку "3. Payments", оплатить всё разом
4. Скачать PDF для бухгалтерии
5. Снять метку **3. Payments**
6. Добавить **Propria GmbH/Processed Invoices** (если Propria) или просто архивировать

## Правило

Не обрабатывать инвойсы по одному в течение дня — только в платёжный слот.
