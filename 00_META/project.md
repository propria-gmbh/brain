# brain

## Назначение

Экзоскелет мышления: усилить интеллект пользователя через структурированный когнитивный pipeline (expert panel, adversarial review, loop interview, pattern detection), сохранить все решения и знания в версионируемом markdown + git, связать внешние рабочие проекты и инструменты (SingularityApp, Google Calendar) в единую систему.

## Что здесь ЕСТЬ

- Стратегические решения (ADR) и журнал мышления
- Ментальные модели, глоссарий, онтология рабочих доменов
- Реестр внешних проектов (08_PROJECTS/) со статусами и целями
- Шаблоны для типовых документов
- Синхронизация задач и встреч (в планах — tools/)

## Чего здесь НЕТ

- Production-код (живёт в отдельных Claude Code репо)
- Секреты, ключи, токены
- Большие бинарные данные
- Ephemeral WIP (короткоживущие рабочие заметки)

## Domains в фокусе

- **E-commerce** (главное рабочее): листинги, реклама, склад, качество товаров, актуализация контента
- **Стратегия и рефлексия**: личные решения, распределение фокуса между проектами, выявление повторяющихся ошибок
- **Исследования**: паттерны, фреймворки, новые инструменты
